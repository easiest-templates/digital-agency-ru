# Vortex Digital - Awwwards-style Landing Page

![Preview](https://cdn.poehali.dev/templates/awwwards-site-ru/preview-ru.jpg)

## О шаблоне

Современный лендинг в стиле Awwwards для digital-агентства. Премиальный дизайн с анимациями, темная тема, интерактивные элементы.

## Технологии

- React 18
- Vite
- Tailwind CSS
- Framer Motion (анимации)
- Lucide React (иконки)

## Установка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Особенности

- Адаптивный дизайн
- Плавные анимации при скролле
- Современная типографика
- Темная тема
- Оптимизированная производительность
