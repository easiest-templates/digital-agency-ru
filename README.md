# Vortex Digital — Сайт креативного агентства

![Preview](https://cdn.poehali.dev/templates/awwwards-site-ru/preview-ru.jpg)

Это готовый шаблон сайта для креативного или digital-агентства. Сайт выполнен в современном темном стиле с эффектными анимациями и интерактивными элементами. Такой дизайн часто можно встретить на премиальных сайтах-победителях Awwwards.

На главной странице вас встречает большой заголовок с анимированной 3D-фигурой на фоне — это создает вау-эффект и сразу привлекает внимание посетителей. Все элементы плавно появляются при прокрутке страницы, что делает просмотр сайта приятным и запоминающимся.

Шаблон включает все необходимые секции: шапку с навигацией, эффектный главный экран, галерею проектов с красивыми эффектами при наведении, превью блога и форму обратной связи. В подвале собраны ссылки на социальные сети и контактная информация.

Отлично подойдет для дизайн-студий, маркетинговых агентств, веб-разработчиков и всех, кто хочет произвести сильное первое впечатление на клиентов. Текст уже на русском языке — просто замените его на свой и добавьте свои проекты в портфолио.

## Установка

```bash
npm install
npm run dev
```

## Технологии

- React + TypeScript
- Three.js (3D)
- GSAP + Framer Motion
- Tailwind CSS
- Vite
