# PixelCraft — Сайт digital-агентства

![Preview](https://cdn.poehali.dev/templates/digital-agency-ru/preview-ru.jpg)

Это готовый шаблон сайта для digital-агентства или веб-студии. Современный темный дизайн с яркими акцентами выглядит профессионально и запоминается с первого взгляда.

На главной странице расположен эффектный заголовок с призывом к действию. Ниже — секция с услугами, где можно показать все направления работы агентства: веб-разработку, дизайн, мобильные приложения и другие.

Портфолио представлено красивыми карточками проектов с описанием и результатами. Есть секция с отзывами клиентов и статистикой компании. В конце страницы — форма обратной связи и футер с контактами.

Шаблон идеально подходит для дизайн-студий, маркетинговых агентств и фрилансеров. Все тексты на русском языке — просто замените их на свои и добавьте свои проекты.

## Установка

```bash
npm install
npm run dev
```

## Технологии

- React + TypeScript
- Tailwind CSS
- Vite
